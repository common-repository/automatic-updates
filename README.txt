Automatic Updates allows admin to manage the automatic updates of the core wordpress and the plugins.
The plugins also disables core update email notification by default. If you need email notification, please enabled it from the plugin option.

== Description ==
<h3>Automatic Updates</h3> allows admin to manage the automatic updates of the core wordpress and the plugins.
The plugin will also allow admin to toggle email notifications with wordpress updates.

== Installation ==
1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the `wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress.
